# AI Pathfinder & Replay Export

This mod provides an **AI-powered pathfinding engine** that simulates Geometry Dash 2.2 physics to find a safe path through any level.

## Features
- **240 TPS Physics Simulation** supporting all 2.2 gamemodes
- **Asynchronous computation** - no UI freezing on Android
- **GDR2 Export** compatible with Eclipse Menu and other replay players
- Supports Cube, Ship, Ball, UFO, Wave, Robot, Spider, and Swing
- Handles slope physics, speed portals, gravity portals, and warp portals

## Usage
1. Open any level's info page
2. Click **"Run Pathfinder"**
3. Wait for analysis to complete
4. Click **"Save .gdr2"** to export the replay
